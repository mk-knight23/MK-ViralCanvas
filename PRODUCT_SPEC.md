# Product Specification - MK ViralCanvas

- **Core Problem:** Users need a simple, fast, free, registration-free utility to solve: "Social image, meme and content creator".
- **Target Audience:** Everyday internet users, developers, and professionals.
- **Key Workflows:**
  1. Instant utility access on page load (simple first).
  2. Advanced configuration options via drawers/tooltips (progressive disclosure).
  3. Safe localStorage or IndexedDB persistence.
- **Features Included:** Upload images, add shapes/text, apply filters, customize canvas template layouts, export PNG/JPG.
